# VFIX: Nanite Ship Upgrade Fix by ThePresident44
# TL;DR:
# Fixes a vanilla bug with Nanite Swarmers/Interdictors where different ship variants would get "upgraded" into the same variant (e.g. Torpedo, Carrier and Gunship Swarmers all "upgrade" into the latest PD variant)
# --------------------
# This mod fixes the hull definition used by Nanite Swarmers and Nanite Interdictors so the game evaluates their upgrade targets correctly.
# The game apparently evaluates what counts as an "upgrade" partially based on the resources used by a ship, which breaks when your ship doesn't use any energy as upkeep.
# To get around this Bio ships have an "evaluation_resource = food" flag set, so that the game doesn't get confused by all ships using the same 0 Energy. For Nanite ships the devs seemingly forgot to set the equivalent "evaluation_resource = nanites" flag.
# Nanite Swarmer
